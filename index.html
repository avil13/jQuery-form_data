How to use?

It's simple.
Connect this plugin to your page: 

            <script src="/path/to/numeric_format.js"></script>

And select element what you want to format. 

            $('p.nead-formatting').numeric_format();

This plugin has some options.

option default
thSep space
dcSep ,
watch FALSE 

$('p.nead-formatting').numeric_format({
                         thSep:" ",
                         dcSep:",",
                         watch:TRUE
                       });
